# Obsidian AI Exporter

Google Gemini、Claude AI、ChatGPT、Perplexity の AI 会話をワンクリックで Obsidian にエクスポート。

## 機能

✨ **マルチプラットフォーム対応** - Gemini、Claude、ChatGPT、Perplexity からエクスポート
📦 **複数の出力オプション** - Obsidian への保存、ファイルダウンロード、クリップボードへコピー
📝 **Obsidian コールアウト** - [!QUESTION] と [!NOTE] で美しく整形
🔬 **Deep Research & Artifacts** - Gemini Deep Research と Claude Extended Thinking レポートを保存
🔄 **追記モード** - 新しいメッセージのみを既存のノートに追加
📋 **YAML フロントマター** - タイトル、ソース、URL、日付、タグなどのメタデータ
📂 **プラットフォーム別整理** - `{platform}` テンプレートで AI ソース別に自動分類
🔃 **自動スクロール** - Gemini の長い会話で全メッセージを自動読み込み
⚙️ **カスタマイズ可能** - 保存先パス、メッセージ形式、フロントマターフィールドなどを設定
🌐 **多言語対応** - 英語・日本語 UI をサポート

## 必要なもの

- [Obsidian](https://obsidian.md/) と [Local REST API](https://github.com/coddingtonbear/obsidian-local-rest-api) プラグイン

## 使い方

1. この拡張機能をインストール
2. Obsidian で Local REST API プラグインを有効化
3. プラグイン設定から API キーをコピー
4. 拡張機能アイコンをクリックして設定を構成
5. 対応 AI プラットフォームで会話を開いて「Sync」をクリック

## 対応プラットフォーム

- **Google Gemini** (gemini.google.com) - 会話と Deep Research レポート
- **Claude AI** (claude.ai) - 会話、Extended Thinking、インライン引用付き Artifacts
- **ChatGPT** (chatgpt.com) - 会話（カスタム GPT を含む）
- **Perplexity** (www.perplexity.ai) - 会話

## プライバシー

- データ収集やアナリティクスなし
- すべてのデータはお使いの端末に保存
- 通信先は localhost（あなたの Obsidian）のみ
